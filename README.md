# Sandwich Order Form

A simple HTML form for submitting sandwich orders.  
The project includes:

- Bread selection (radio buttons)
- Toppings selection (checkboxes)
- Delivery options
- Telephone and quantity fields
- Branch selection
- Basic responsive CSS styling

## How to Use
1. Fill in all required fields.
2. Select your preferred bread, toppings, and delivery option.
3. Enter your telephone number and number of sandwiches.
4. Choose a branch.
5. Click **Place Order** to submit the form.

## Technologies
- **HTML5**
- **CSS3**

## Purpose
This project is a basic exercise for practicing:
- Form structure
- Input types
- Labels and accessibility
- Styling forms with CSS

## Preview
Open `index.html` in your browser to view the form.
